## Introduction

ARAdmin is a comprehensive political party management system designed specifically for Affirmative Action (AR). This platform provides tools for managing members, organizing campaigns, communicating with supporters, handling finances, and coordinating party activities across Namibia.

### Key Features

- Member registration and approval system
- Campaign planning and budget management
- Multi-channel broadcasting (Posts, SMS)
- Financial tracking and reconciliation
- Polls and voting
- Event calendar management
- Community organization
- Mobile app integration

## Getting Started

### Accessing the System

1. Navigate to your ARAdmin URL in a web browser
2. You will be presented with the login page

### First-Time Login

**For Administrators:**
1. Click on "Sign Up" if you don't have an account
2. Enter your details:
   - Full Name
   - Email address
   - Password (minimum 6 characters)
3. Click "Sign Up"
4. Your account will be created with basic access
5. Contact the super administrator to upgrade your account to admin level

**Logging In:**
1. Enter your registered email address
2. Enter your password
3. Click "Sign In"
4. You will be redirected to the Dashboard

## Member Management

The Members section helps you manage party membership applications and member data.

### Viewing Members

1. Click on "Members" in the sidebar
2. You'll see a list of all registered members
3. Use the search bar to find specific members by:
   - Name
   - Email
   - Phone number
   - Region

### Member Information Displayed

Each member card shows:
- Full name and membership number
- Region and phone number
- Application date
- Current status (Review, Approved, Rejected)

### Adding a New Member Manually

1. Click the "Add Member" button
2. Fill in the required fields (marked with *):
   - First Name *
   - Surname *
   - ID Number *
   - Phone Number *
   - Gender *
   - Home Language *
   - Region *
3. Fill in optional fields:
   - Email
   - Date of Birth
   - Occupation
   - Constituency
   - Town/City
   - Village/Suburb
   - PO Box
   - Physical Address
4. Click "Add Member"
5. The member will be automatically approved

### Reviewing Member Applications

Members who register through the mobile app appear with "Review" status.

**To Approve a Member:**
1. Click on the member card to view details
2. Review all provided information including uploaded ID card
3. Click "Approve Application"
4. A membership number will be automatically generated (format: AR-0001, AR-0002, etc.)
5. The member will receive a notification about approval

**To Decline a Member:**
1. Click on the member card
2. Click "Decline Application"
3. The member's application will be marked as rejected
4. A notification will be sent to the member

### Viewing Member Details

Click on any member to view comprehensive information:
- **Uploaded ID Card:** View the ID document uploaded by the member
- **Personal Information:** ID number, date of birth, gender, language, occupation
- **Contact Information:** Email, phone number
- **Address Information:** Region, constituency, town, village, PO Box, physical address
- **Volunteer Information:** Willingness to volunteer, interests, availability, leadership interest

### Exporting Member Data

1. Click the "Export" button at the top
2. A CSV file will be downloaded with all member information
3. File naming format: `members_YYYY-MM-DD.csv`

### Member Statistics

View analytics on:
- **Members by Region:** Bar chart showing top 5 regions by membership
- **Age Distribution:** Pie chart showing member demographics (18-25, 26-35, 36-50, 51+)

## Campaign Management

Organize and track political campaigns across regions.

### Creating a New Campaign

1. Click "Campaigns" in the sidebar
2. Click "Create Campaign"
3. Fill in campaign details:
   - **Campaign Name:** Descriptive title
   - **Description:** Campaign objectives and details
   - **Start Date:** Campaign launch date
   - **End Date:** Campaign conclusion date
   - **Region:** Target region (or select "All Regions")
   - **Status:** Draft, Planning, Active, or Completed
4. Click "Create Campaign"

### Managing Campaign Details

**Accessing Campaign Details:**
1. Click on any campaign card
2. View comprehensive campaign information

**Campaign Information Panel:**
- Campaign name and description
- Duration (start and end dates)
- Target region
- Current status
- Progress tracking

### Campaign Images

**Uploading Campaign Images:**
1. Open campaign details
2. Scroll to the "Campaign Images" section
3. Click "Upload Images"
4. Select one or multiple images
5. Images will be displayed in a gallery
6. Click "Delete" on any image to remove it

**Image Guidelines:**
- Supported formats: JPG, PNG
- Recommended size: 1920x1080px or smaller
- Maximum file size: 5MB per image

### Campaign Tasks

**Creating Tasks:**
1. In campaign details, find the "Tasks" section
2. Click "Add Task"
3. Enter task details:
   - Task title
   - Description
   - Due date
   - Assigned collaborator
4. Click "Save"

**Managing Tasks:**
- Mark tasks as complete with checkbox
- Edit task details
- Delete completed or unnecessary tasks
- Track task progress

### Campaign Collaborators

**Adding Collaborators:**
1. In campaign details, find "Collaborators" section
2. Click "Add Collaborator"
3. Select user from dropdown
4. Choose role: Coordinator, Volunteer, or Organizer
5. Click "Add"

**Collaborator Roles:**
- **Coordinator:** Full campaign management access
- **Organizer:** Can manage tasks and events
- **Volunteer:** Can view and update assigned tasks

### Campaign Budget Management

1. Click "Budget" in campaign details
2. View budget overview:
   - Total budget allocated
   - Total expenses
   - Remaining budget
3. Add budget items:
   - Item name
   - Amount
   - Category
   - Notes
4. Track expenses in real-time

### Campaign Attendance Tracking

**Recording Event Attendance:**
1. Go to campaign details
2. Click "Attendance" tab
3. Click "Record Attendance"
4. Select or add attendee information:
   - Name
   - Contact details
   - Region
5. Save attendance record

**Viewing Attendance:**
- See total attendees
- Export attendance lists
- Filter by date or region

## Broadcasting System

Communicate with members through posts and SMS.

### Broadcasting Posts

**Creating a Post:**
1. Navigate to "Broadcasting" → "Enhanced Broadcasting"
2. Click "Create New Post"
3. Fill in post details:
   - **Title:** Post headline
   - **Content:** Message body (supports rich text)
   - **Media:** Upload images or videos
4. Choose audience:
   - All members
   - Specific region
   - Specific community
5. Choose posting time:
   - Post now
   - Schedule for later
6. Click "Publish" or "Schedule"

**Managing Posts:**
- View all published posts
- See engagement metrics (views, likes, comments)
- Edit or delete posts
- Pin important posts to top

**Comments and Engagement:**
- Members can comment on posts via mobile app
- View and moderate comments
- Respond to member feedback

### SMS Campaigns

**Creating an SMS Campaign:**
1. Navigate to "Broadcasting" → "SMS Campaigns"
2. Click "Compose" tab
3. Enter campaign details:
   - **Campaign Title:** Internal reference name
   - **Message:** SMS content (160 characters recommended)
4. Select recipients:
   - **All Members:** Send to entire membership
   - **Individual Member:** Search and select one member
   - **By Region:** Select specific region
   - **By Status:** Filter by membership status
5. View estimated recipient count
6. Choose action:
   - **Save Draft:** Save for later
   - **Schedule:** Set future send time
   - **Send Now:** Immediate delivery

**Character Count Guidelines:**
- SMS messages over 160 characters will be sent as multiple messages
- Character counter shows total length and overage
- Cost increases with message segments

**SMS Campaign History:**
1. Click "Campaigns" tab
2. View all sent campaigns:
   - Campaign title and message
   - Send date/time
   - Recipient count
   - Success/failure rates
3. Track delivery status

**Scheduling SMS Campaigns:**
1. Click "Schedule" button
2. Select date and time
3. Campaign will be added to calendar
4. Review scheduled campaigns in Calendar section

## Best Practices

### Member Management
- Review applications within 24-48 hours
- Verify ID documents before approval
- Keep member records up to date
- Regularly export member data for backup

### Campaign Organization
- Create campaigns early with clear objectives
- Assign responsible coordinators
- Track budgets carefully
- Document all activities
- Collect attendance records

### Communication
- Schedule broadcasts during optimal times
- Keep SMS messages concise (under 160 characters)
- Use appropriate channels for message type
- Monitor engagement metrics
- Respond to member feedback

### Financial Management
- Record all transactions immediately
- Upload proof of payment for all donations
- Reconcile accounts weekly
- Generate monthly financial reports
- Maintain proper documentation

### Data Security
- Change passwords regularly
- Log out when finished
- Don't share admin credentials
- Review user access periodically
- Back up data regularly

## Troubleshooting

### Common Issues

**Cannot Log In:**
1. Verify email and password are correct
2. Check for caps lock
3. Try password reset
4. Clear browser cache
5. Contact administrator if issue persists

**Member Registration Not Appearing:**
1. Check "Review" status filter
2. Refresh the page
3. Verify member completed registration on mobile app
4. Check internet connection

**SMS Not Sending:**
1. Verify recipient has valid phone number
2. Check SMS credits/balance
3. Ensure SMS gateway is configured
4. Contact super administrator

**Images Not Uploading:**
1. Check file size (max 5MB)
2. Verify file format (JPG, PNG)
3. Check internet connection
4. Try smaller file size
5. Refresh page and try again

**Financial Records Not Saving:**
1. Ensure all required fields are filled
2. Check for valid numeric values
3. Verify date format
4. Upload supporting documents
5. Refresh and try again

## Glossary

**Admin:** System administrator with elevated permissions

**Broadcast:** Message sent to multiple members simultaneously

**Campaign:** Organized political activity with specific goals

**Collaborator:** User assigned to help with campaign activities

**Community:** Regional or constituency-based member organization

**Constituency:** Electoral district within a region

**Coordinator:** User responsible for managing specific area or activity

**Dashboard:** Main overview screen with key statistics

**Membership Number:** Unique identifier assigned to approved members (format: AR-####)

**Poll:** Survey or voting mechanism for member input

**Region:** One of Namibia's 14 administrative regions

**RLS (Row Level Security):** Database security ensuring users see only their authorized data

**SMS Campaign:** Text message sent to members via SMS

**Status:** Current state of member application or campaign (Draft, Active, Completed, etc.)

**Super Admin:** Highest level administrator with full system access
